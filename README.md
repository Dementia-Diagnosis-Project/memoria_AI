# MEMORIA_AI

![Award](img/award.jpg)

## 프로젝트 소개
- 치매 진단을 목표로 다양한 딥러닝 모델을 적용
- Django, Web 개발, Figma를 활용하여 AI 팀과 UX/UI 팀이 협업한 종합 프로젝트
- 라틴어로 '기억(Memoria)'을 의미, 기억을 지키기 위한 AI 개발
